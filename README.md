# React + Node Monorepo

✨ **Monorepo generated with [Nx](https://nx.dev)** ✨

## Instalation

Run `npm install`

## Running

Open a terminal and run:

- `npm run serve-api`

In another a terminal and run:

- `npm run serve-frontend`

## Testing

- `npm run test-api`
- `npm run test-frontend`
